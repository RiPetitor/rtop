# ASCII‑логотипы

Отрисовка логотипов для System вкладки находится в `src/ui/system/logo.rs`.

## Формат

- Логотип — массив ASCII‑строк.
- Цветовые токены: `$1`, `$2`, ... `$9` внутри строки.
- `$0` сбрасывает цвет к дефолтному accent.
- Токены не печатаются, они только меняют цвет последующего текста.

Пример строки:

```
"  $1/\\$2__"
```

## Как добавить новый логотип

1) Добавьте массив `LOGO_<NAME>` со строками.
2) Добавьте палитру `PALETTE_<NAME>` с цветами под `$1..$N`.
3) Добавьте `LOGO_SPEC_<NAME>`.
4) Обновите `select_logo()` под свой OS ID или имя.

На Linux используется `/etc/os-release` (`ID` или `NAME`).
На остальных платформах — `System::name()`.

## Примечания

- Держите строки короткими — иначе будет обрезка в узких терминалах.
- Используйте только ASCII, чтобы не ломать ширины в TUI.
